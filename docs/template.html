Template of game

main.cpp, fly_engine_template.hpp and game_main.hpp - files for game template.
You can leave main.cpp as is, only change include of fly_engine_template.hpp.
fly_engine_template.hpp - header of projects, it includes libraries, files of game, engine.hpp, declare global variables, objects and make defines.
game_main.hpp - handle init and exit from game, you can add your custom init / exit here. Also it has frame(), dont touch it, and tick(), all updates work here. It update control via SDL_Event, handle game state via if or switch and make timed updates (engine.tick()) using timers.
